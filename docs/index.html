üì¶ Database - MySqlTable

The Snake\Database\MySql\Table class allows you to define and interact with your MySQL tables using a fluent, expressive syntax.

use Snake\Database\MySql\Table;

$table = new Table('users');
$table->addColumn('id', 'int')->primary();
$table->addColumn('name', 'string');
$table->addColumn('email', 'string')->unique();
$table->create();

This will generate a new users table with the specified schema. You can also modify, drop, or query tables fluently.
